News Module : 

    A- News Posts

        1 - Adding a News Post : 

            Route --> http://127.0.0.1:8000/news/add
            Priviliges --> Admin & HR
            TYPE : POST

            BODY : 
            {
            "title": "AI Will Change the Future",
            "content": "OpenAI has released powerful new toolsss."
            }

            EXEPECTED RESPONSE : 
            {
            "news_uid": "NEWS-88205977",
            "title": "AI Will Change the Future",
            "content": "OpenAI has released powerful new toolsss.",
            "posted_at": "2025-06-09 05:06 PM",
            "author": "admin2",
            "id": 71
            }

        
        2 - Deleting a News Post :
            
            Route --> http://127.0.0.1:8000/news/delete/12
            Priviliges --> Admin & HR
            TYPE : GET

            BODY : N/A

            EXEPECTED RESPONSE : 
            {
            "detail": "News deleted successfully"
            }

    B- Comments on News Posts
        
        3 - Posting a Comment on a News Post :
            
            Route --> http://127.0.0.1:8000/comment/post
            Priviliges --> All
            TYPE : POST

            BODY :
            {
            "news_id": 4,
            "content": "Great update! Thanks."
            }

            EXEPECTED RESPONSE : 
            {
                "news_id": 4,
                "user_id": 7,
                "content": "Great update! Thanks.",
                "commented_at": "2025-06-09 05:47 PM",
                "id": 17
            }

        
        4 - Deleting a Comment from a News Post : 

            Route --> http://127.0.0.1:8000/comment/delete/5
            Priviliges --> Admin & Comment Owner
            TYPE : Delete

            BODY : N/A

            EXEPECTED RESPONSE :

            IF(Authorized == TRUE)
            {
                "detail": "Comment deleted successfully"
            }

            OR

            ELSE
            {
                "detail": "Permission denied"
            }
